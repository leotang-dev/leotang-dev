# Leo Tang

Founder of **GlyphEvo Labs** — building open-source tools for AI agents, code intelligence, and context engineering.

## What I’m building

**GlyphEvo Labs** focuses on developer-oriented infrastructure that helps AI agents understand repositories, preserve useful context, and work more safely across complex software systems.

Current project directions:

* **ArcGraph** — codebase graph analysis and repository understanding for AI agents.
* **TraceMind** — agent memory, task traces, and context continuity.
* **GlyphEvo** — the main brand and future platform home for agent-native developer tools.
